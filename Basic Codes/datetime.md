# datetime
```python
import datetime
```
## datetime.datetime
```python
datetime.datetime(2018, 5, 19)
```
- \>\>\> datetime.datetime(2018, 5, 19, 0, 0)
### datetime.datetime.now()
```python
datetime.datetime.now()
```
- \>\>\> datetime.datetime(2020, 8, 18, 21, 44, 20, 835233)
## timestamp()
```python
datetime.datetime.now().timestamp()
```
- 1970년 1월 1일 0시 0분 0초로부터 몇 초가 지났는지 출력.
